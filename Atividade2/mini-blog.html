Personalização de parágrafo com fonte específica

A modificação da fonte de uma texto específico em uma página HTML é feita a partir dos seguintes passos:

  • Escolher uma fonte e obter os links necessários, por meio de sites como Google Fonts;
  • Adicionar o(s) link(s) na tag "head";
  • Aplicar a fonte ao elemento desejado usando CSS, seja por meio de uma classe ou diretamente no seletor do elemento.

Aplicação de diferentes estilos de uma mesma fonte na página HTML

Espessura:

Para aplicar diferentes espessuras de uma mesma fonte em uma página HTML, siga os seguintes passos:

  • Escolha uma fonte que ofereça múltiplsas espessuras (normal, negrito, itálico, etc.) e obtenha os links necessários, por meio de sites como o Google Fonts, já mencionado acima;
  • Adicione os links na tag "head";
  • Crie classes CSS para cada espessura desejada, especificando a propriedade "font-weight" para negrito e "font-style" para itálico, por exemplo;
  • Aplique as classes aos elementos HTML conforme necessário para exibir as diferentes espessuras da fonte.

Exemplo de aplicação de diferentes estilos da fonte "Playfair Display":

  • Este parágrafo utiliza o estilo normal da fonte "Playfair Display".
  • Este parágrafo utiliza o estilo negrito da fonte "Playfair Display" com "weight" = 700.
  • Este parágrafo utiliza o estilo negrito da fonte "Playfair Display" com "weight" = 900.

Inclinação:

Para aplicar valores diferentes de inclinação na sua fonte, os passos são semelhantes aos de espessura, veja:

  • Com a fonte já definida, como foi explicado acima, crie classes para cada inclinação desejada.
  • Com as classes definidas, especifique no CSS a inclinação de cada classe, por meio da propriedade "font-style". Você pode utilizar, por exemplo, "deg" que é graus, ou seja, você vai definir quantos graus de inclinação você quer pra letra.

  • Este parágrafo utiliza "font-style = 20deg" na fonte "Roboto Flex".

Escolha da melhor imagem

As três resoluções no tópico acima, se baixadas com uma internet de menor desempenho, levam tempos distintos e talvez significativos, para otimizar isso, existem tags como "<Picture>". Também tem o srcset, que não é uma tag mas deixa o processo de escolha automático escolhendo a melhor resolução com base no tamanho da tela. Nesse caso em que as imagens são as mesmas e resoluções diferentes, o srcset desempenha melhor. O picture é mais recomendado quando tem que trocar para outra imagem por motivos de tamanho, por exemplo: na respectiva tela a imagem x ficará cortada, então coloca a imagem y. Abaixo utilizo o "<Picture>" apesar de trabalhar com a mesma imagem. É só porque eu quero mesmo.
