# 📚 Atividades de Programação para Internet

Repositório criado para armazenar as atividades desenvolvidas na disciplina de Programação para Internet.

## 🚀 Tecnologias utilizadas
- HTML5
- CSS3
- JavaScript

## 📂 Estrutura do repositório
As atividades estão organizadas em pastas:

- 📁 Atividade1
- 📁 Atividade2
- 📁 Atividade3

Cada pasta contém os arquivos referentes à atividade proposta na semana pelo professor.

## 🎯 Objetivo
Praticar conceitos fundamentais de desenvolvimento web, como:
- Estruturação de páginas com HTML
- Estilização com CSS
- Interatividade com JavaScript
